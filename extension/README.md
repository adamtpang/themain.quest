# the main quest, Chrome extension

Your gamified life command center, lifeleft style, pinned to the toolbar.

- The **badge** on the icon always shows your years of the game remaining, no click needed, refreshed daily.
- Click it to open the **command center**: your life-left clock (days remaining, percent of the game spent), your **live main quest, level, streak, and boss HP** pulled straight from themain.quest, a rotating identity, and a lesson from one of the **greatest lives ever lived** with a move to run today. Hit "enter the match" to jump into the game.

The life clock is computed locally from your birthdate, so it works offline. The live game state is mirrored from your own themain.quest tab into local storage, so nothing leaves your machine.

## Install (load unpacked)

1. Open `chrome://extensions` in Chrome (or Edge, Brave, Arc, any Chromium browser).
2. Turn on **Developer mode** (top right).
3. Click **Load unpacked**.
4. Select this `extension` folder.
5. Click the puzzle-piece icon in the toolbar, then the pin next to "the main quest" to keep it visible.
6. Open **themain.quest** once so the command center can sync your live quest.

Already loaded an older version? Hit the reload arrow on the extension card in `chrome://extensions` to pick up the new command center.

## Files

- `manifest.json`, Manifest V3, permissions `alarms` (daily badge) and `storage` (the live quest sync).
- `background.js`, paints the years-left badge on the icon and refreshes it daily.
- `bridge.js`, a content script on themain.quest that mirrors your game state (main quest, level, streak, boss) into local storage for the popup. Reads only, never writes to the game.
- `popup.html` / `popup.js`, the command center: life clock, live stats, the greatest-lives mentors, and the enter-the-match button.
- `icon16.png` / `icon48.png` / `icon128.png`, the pixel heart, regenerated by `node gen-icons.mjs`.

## Publishing to the Chrome Web Store (optional, later)

Load-unpacked works for you today. To make it installable by others you would zip this folder, pay the one-time $5 Chrome Web Store developer fee, and submit it for review. Not needed for personal use.
